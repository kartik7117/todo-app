# Todo App

A simple Todo application built using **Spring Boot** and **MySQL** that allows users to manage their daily tasks.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML / CSS

  ├── controller
  ├── model
